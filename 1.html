Hypertext Markup Language (HTML) is the standard markup language for creating web pagesand web applications.

strong = 글씨 진하게

u = 밑줄

br = h1태그는 열리는 닫히는 태그가 쌍으로 존재하지만 br태그는 줄바꿈의 시각적인 의미만 가지고있음(태그를 닫지않음) = new line tag(br태그)

margin-top:40px; = 단락위를 40픽셀만큼 떨어트린다.

ol과 ul의 차이는 ol은 Ordered List, ul은 Unordered List
ul은 비정렬 목록 태그 : 순서가 중요하지 않은 목록을 나타낼때 사용
ol은 정렬 목록 태그 : 순서가 중요한 목록을 나타낼때 사용
li은 목록 항목 태그 : 목록안의 아이템
title : 제목
h1 = h1 ~ h6 글자크기 h1 → h6로 갈수록 작아짐(제목태그(줄바꿈도됨))
